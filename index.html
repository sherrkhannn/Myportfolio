MeowMeow(Dating App for cats) Project

Created a cat-themed dating website with HTML and CSS involved balancing whimsy and usability. I structured profiles using CSS Flexbox ,Grid and Bootstrap for responsive layouts, ensuring compatibility across devices. Designing playful yet intuitive buttons and cards required careful color contrasts and rounded corners for a friendly vibe. Handling hover effects added interactivity without JavaScript. A key challenge was to maintain the Responsiveness while maintaining visual harmony. Media queries fixed mobile alignment issues, and testing revealed font readability problems, solved with Google Fonts. The project blends feline charm with user-centric design, proving HTML/CSS’s versatility for creative, functional interfaces.

Mondrian Painting Project

Created a Mondrian-inspired painting with HTML and CSS posed challenges in balancing structure and abstraction. I used CSS Grid to replicate his geometric precision, adjusting grid-template-columns/rows and spans to align blocks. Managing color blocks required careful class assignments to primary hues and white/black backgrounds. Responsiveness was tricky—viewport settings and fractional units ensured proportions stayed intact across screens. Thick borders overlapped grid gaps, so I fine-tuned border properties and box-sizing to maintain clean lines. Testing revealed alignment issues, which I fixed by recalculating grid divisions. The result honors Mondrian’s De Stijl style, blending rigid grids with dynamic color blocks, proving CSS’s power for artistic expression.
